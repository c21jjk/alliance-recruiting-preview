# Alliance Recruiting — Preview

Prototype of the rebuilt Recruiting section for AllianceDashAI.

## Deploy to Vercel (iPad flow)

1. Create a new GitHub repo
2. Upload all these files to the repo root
3. Go to vercel.com → New Project → Import this repo
4. Vercel auto-detects Vite. Click Deploy.
5. You get a live URL in ~60 seconds.

## Updating the prototype

Edit `src/App.jsx` directly in GitHub's web editor (tap the pencil). Commit. Vercel redeploys automatically. Refresh the iPad Safari tab.

## Add to iPad home screen

Open the Vercel URL in Safari → Share → Add to Home Screen. Opens full-screen like a native app.
